## 📌 진행 방법
1. Easy → Medium 순서대로 한 주제씩 집중
2. 깃허브에 문제 풀이 코드 업로드 (폴더 구조: `category/문제명.java`)
3. 매주 5~7문제 목표 → 한 달에 최소 20문제 완료
4. 실패한 문제는 2~3일 뒤 재풀이 (Spaced Repetition)
